    рЃАрЃљрЃЎрЃЮрЃюрЃбрЃљрЃЦрЃбрЃЮ рЃцрЃЮрЃарЃЏрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ, рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА, рЃарЃЮрЃЏ рЃарЃЮрЃњрЃЮрЃарЃф рЃЊрЃўрЃќрЃљрЃўрЃюрЃерЃўрЃљ, 
    рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃљрЃа рЃљрЃЦрЃЋрЃА рЃБрЃцрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃбрЃЮрЃЋрЃЮрЃА рЃфрЃљрЃарЃўрЃћрЃџрЃў/рЃерЃћрЃБрЃЋрЃАрЃћрЃЉрЃћрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЋрЃћрЃџрЃћрЃЉрЃў 
    рЃЊрЃљ рЃАрЃЦрЃћрЃАрЃўрЃА рЃЏрЃЮрЃАрЃљрЃюрЃўрЃерЃюрЃў рЃАрЃћрЃЦрЃфрЃўрЃљ, рЃЌрЃБрЃЎрЃў рЃфрЃљрЃарЃўрЃћрЃџрЃА рЃЊрЃљрЃбрЃЮрЃЋрЃћрЃЉрЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃБрЃбрЃљрЃюрЃЮрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў
